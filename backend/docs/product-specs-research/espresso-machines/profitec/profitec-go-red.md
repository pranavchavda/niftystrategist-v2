# Profitec GO Red Espresso Machine - Technical Specifications

## Overview
The Profitec GO Red is a compact single boiler espresso machine designed for fast heat-up times and consistent espresso extraction. This is the red color variant of the Profitec GO series.

## Boiler System
- **Type**: Single boiler (NOT thermoblock)
- **Material**: Insulated brass boiler
- **Capacity**: 0.3 liters (300ml)
- **Heating Element**: Copper heating element
- **Warm-up Time**: 5-7 minutes (Fast Heat-Up Mode)

## Power Specifications
- **EU/UK/CH/AU**: 230V, 1300W
- **US**: 115V, 900W

## Pump System
- **Type**: Vibration pump
- **Features**: 
  - Pump pressure gauge
  - Expansion valve for brewing pressure adjustment

## Dimensions & Weight
- **Width**: 210 mm (8.27")
- **Depth**: 362 mm without portafilter / 410 mm with portafilter
- **Height**: 381 mm (15")
- **Weight**: 12.9 kg (28.4 lbs)

## Water System
- **Water Tank Capacity**: 2.8 liters
- **Water Tank**: Removable with lid

## Key Features
- **PID Temperature Control**: Digital temperature control for precise brewing
- **Shot Timer**: Built-in brewing counter in seconds
- **ECO Mode**: Programmable energy-saving mode
- **Steam System**: Rotary valve for steam extraction
- **Group Head**: Ring brew group with 3-way solenoid valve
- **Portafilter**: 58mm commercial size

## Control Features
- PID display with temperature readout
- Shot timer
- ECO mode programming
- Temperature adjustment

## Build Quality
- Commercial-grade components
- Brass boiler with insulation for temperature stability
- Stainless steel and powder-coated steel construction

## Notes
- All color variants (Black, Red, Yellow, Blue) share identical specifications
- Compact footprint ideal for smaller kitchens
- Fast 5-7 minute heat-up time makes it convenient for daily use
- Single boiler design requires temperature surfing between brewing and steaming
