# Markdown Quick Reference for CMS Editors

## Basic Formatting

### Headings
```markdown
### Main Heading (Level 3)
#### Sub Heading (Level 4)
##### Minor Heading (Level 5)
```

### Text Styling
```markdown
**Bold text**
*Italic text*
***Bold and italic text***
```

### Lists

#### Bullet Lists
```markdown
- First item
- Second item
- Third item
```

#### Numbered Lists
```markdown
1. First step
2. Second step
3. Third step
```

### Paragraphs
Separate paragraphs with a blank line:
```markdown
First paragraph text here.

Second paragraph text here.
```

## Using the Editor

### Toolbar Buttons
- **### (Heading)**: Convert current line to heading
- **B (Bold)**: Make selected text bold
- **I (Italic)**: Make selected text italic
- **• (Bullet List)**: Insert bullet point
- **1. (Numbered List)**: Insert numbered item

### Live Preview
- Click the **eye icon** to toggle preview
- Preview updates as you type
- See exactly how your content will look

### Tips
1. **Select text** before clicking formatting buttons to apply formatting
2. **Click buttons** without selection to insert template text
3. **Use keyboard** for faster editing (type markdown directly)
4. **Preview often** to check formatting

## Common Patterns

### FAQ Answer with List
```markdown
The best time to descale your espresso machine depends on several factors:

### Recommended Schedule:

- **Daily use:** Every 2-3 months
- **Weekly use:** Every 4-6 months
- **Occasional use:** Every 6-12 months

Always check your machine's manual for specific recommendations.
```

### Educational Content with Steps
```markdown
Single dosing is a coffee grinding technique where you measure and grind exactly the amount of coffee needed for each brew.

### How to Single Dose:

1. Weigh your coffee beans precisely
2. Pour beans directly into grinder
3. Grind until all beans are processed
4. Use immediately for best results

### Benefits:

- **Freshness:** Keeps beans fresh longer
- **Precision:** Exact dose every time
- **Variety:** Easy to switch coffees
```

## Troubleshooting

### Preview Not Showing
- Make sure the eye icon is highlighted (blue)
- Check that you've entered some content

### Formatting Not Working
- Ensure proper markdown syntax (no spaces inside ** or *)
- Check that lists have proper spacing
- Use blank lines to separate blocks

### Bold/Italic Mixed Up
- `**text**` = **bold**
- `*text*` = *italic*
- `***text***` = ***bold and italic***

## Best Practices

### Do:
- ✅ Use headings to organize content
- ✅ Keep paragraphs concise
- ✅ Use lists for multiple points
- ✅ Preview before saving
- ✅ Use blank lines between sections

### Don't:
- ❌ Mix formatting styles inconsistently
- ❌ Create very long paragraphs
- ❌ Forget blank lines between blocks
- ❌ Use too many heading levels
- ❌ Nest lists (keep them simple)

## Character Count

The editor shows character count in the top-right corner. Use this to:
- Keep content concise
- Match character limits (if any)
- Track content length

## Saving Your Work

1. **Edit** content in the markdown editor
2. **Preview** to verify formatting
3. **Click Save** to save changes
4. **Wait** for confirmation message

Changes are converted to Shopify format automatically!
